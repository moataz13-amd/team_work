Project Overview

Forbidden Area is an advanced intelligent security system designed to monitor and protect restricted zones. The system relies on radar sensing to continuously scan a predefined area and detect any unauthorized movement. When a target enters the forbidden area, the system reacts automatically by identifying, tracking, and responding to the intrusion through multiple connected units.

1. Radar Detection Unit
Continuously scans the forbidden area for moving or static objects.
Calculates the target’s distance and direction.
Confirms detection to reduce false alarms.
Sends real-time alert signals to other system units.
Acts as the main decision-making unit of the system

2. Laser Tracking Platform
Receives target position data from the radar unit.
Uses a servo-controlled mechanism for precise laser alignment.
Automatically tracks and points the laser toward the detected target.
Provides a visual indication of target location.
Operates independently from the mobile unit.

3. Mobile Car Response Unit
Receives alert notifications when an intrusion is detected.
Is manually controlled by a user after alert reception.
Moves toward or around the target area as required.
Represents the system’s physical response element.

4. System Communication & Integration
Wireless communication enables data exchange between units.
Each unit operates independently without direct hardware dependency.
The modular design allows easy expansion or replacement of units.
Ensures system reliability and flexibility.

5. Project Objectives
Detect unauthorized access to forbidden areas.
Provide real-time alerts and visual target identification.
Combine sensing, tracking, and response in one system.
Apply concepts of embedded systems, automation, and IoT.

6. Potential Applications
Smart security and surveillance systems.
Military and defense simulations.
Industrial and facility protection.
Academic research and educational demonstrations.
